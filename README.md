# Authenticator

This is a simple TOTP (Time-based One-Time Password)

## Tech stack

- Bun
- Astro x React
- TailwindCSS
- Shadcn/UI
- Prettier and ESLint

## Self-run

### Requirements

- bun

### Setup environment

```bash
bun install
```

### Run

```bash
bun run dev
```

### Build

```bash
bun run build
```

### Run build

```bash
bun run preview
```
